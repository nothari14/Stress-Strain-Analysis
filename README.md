# 🔩 Stress Concentration Analysis of Plate with Holes using SolidWorks  

## 📌 Project Overview  
This project focuses on analyzing stress concentration in a flat plate with circular discontinuities using SolidWorks Simulation. The objective is to study how the presence of holes affects stress distribution, deformation, and strain when the plate is subjected to tensile loading.

The simulation demonstrates how geometric discontinuities disturb the load path and create localized regions of high stress.

---

## ⚙️ System Description  

The model consists of a rectangular plate with:  
- One central circular hole  
- Two smaller side holes  

### Material:
- Alloy Steel  

### Boundary Conditions:
- One end fixed (Fixed Support)  
- Opposite end subjected to tensile force (1000 N)  

---

## ⚙️ Simulation Setup  
- Static Structural Analysis  
- Solid mesh generated with refinement near hole edges  
- Load applied normal to surface
